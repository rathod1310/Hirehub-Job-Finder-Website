# 💼 HireHub - Job Finder Website

A self-created project developed using the **Django Framework**.
This system connects **Companies** and **Applicants**, allowing companies to post job openings and applicants to explore and apply for suitable positions.

---

## 🚀 Features

### 👔 Company Side

* Register and login securely
* Post new job openings
* View all posted jobs
* View details of applicants who applied
* Logout functionality

### 🙋 Applicant Side

* Register and login securely
* View all available job listings
* Apply for desired jobs
* View applied jobs in profile
* Logout functionality

---

- HTML/CSS/JS
- Python
- Django
